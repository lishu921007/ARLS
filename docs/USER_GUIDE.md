# 小白用户使用说明

## 1. 怎么安装
1. 双击安装包 `ARLS_Setup.exe`
2. 一直点“下一步”
3. 安装完成后桌面会出现快捷方式

## 2. 怎么启动
双击桌面“启动行政复议事项管理系统”
系统会自动：
- 启动后端服务
- 启动前端页面
- 自动打开浏览器

## 3. 怎么登录
默认账号：
- 用户名：admin
- 密码：admin123

## 4. 怎么新增事项
1. 进入“新建事项”
2. 填写通知书编号、申请人、经办部门
3. 点击签收日期的日历控件选择日期
4. 保存

## 5. 怎么看临期 / 紧急 / 超期
1. 进入“临期/紧急/超期”页面
2. 顶部先看三张风险卡：临期、紧急、超期
3. 下面风险清单默认把高风险事项排在前面
4. 可直接点“手工微信”或“详情”继续处理

## 6. 工作日 / 节假日是怎么判断的
系统默认：
1. 先看人工覆盖配置
2. 再按中国法定节假日 / 调休自动判断
3. 最后兜底按周一到周五识别为工作日

说明：
- 一般不需要人工维护
- 只有遇到特殊情况，才在“工作日/节假日”页面新增一条人工覆盖

## 7. 怎么测试微信自动发送
### 先做预览测试（推荐）
1. 进入“系统参数”页面
2. 确认：
   - 自动发送开关 = 开启
   - 预览模式 = 开启
3. 打开 Windows 微信并登录
4. 进入“微信测试”页面
5. 填写你的微信备注名
6. 点击“发送测试”
7. 点击“立即处理一条队列”
8. 去“微信发送日志”页面查看是否成功记录

### 再做真实发送测试
1. 进入“系统参数”页面
2. 把“预览模式”改为关闭并保存
3. 确认电脑微信已登录且停留在可操作状态
4. 回到“微信测试”页面
5. 填写正确的微信备注名
6. 点击“发送测试”
7. 点击“立即处理一条队列”
8. 到微信里确认是否真实收到消息
9. 再到“微信发送日志”页面确认状态

## 8. 自动发送验证 SOP（完整验收）
1. 在“系统参数”中开启自动发送
2. 先保持预览模式开启
3. 新建一条会进入“临期 / 紧急 / 超期”的事项
4. 等待定时扫描，或观察队列是否自动新增任务
5. 在“微信测试”页查看待发送队列是否出现该事项
6. 点击“立即处理一条队列”
7. 查看“微信发送日志”确认链路打通
8. 然后关闭预览模式，再重复一次做真实发送验收

## 9. 微信发不出去怎么办
请依次检查：
- 微信是否已打开
- 微信是否已登录
- 联系人微信备注名是否正确
- 系统参数里是否还开启了预览模式
- 微信窗口标题是否仍是“微信”
- 失败任务是否可在“微信测试”页重试

## 10. 怎么备份数据
进入“备份恢复”页面：
- 点击“一键备份数据库”可立即生成备份
- 点击“恢复此备份”可用已有备份恢复
- 点击“上传数据库并恢复”可导入外部 `.db` 文件进行恢复

注意：
- 每次恢复前，系统会自动备份当前数据库
- 建议恢复前先手工再做一次备份

## 11. 常见问题
- 页面打不开：请重新双击“启动系统”
- 登录失败：检查用户名密码是否正确
- 微信发送失败：先在“微信测试”页做单独测试
- 恢复后数据不对：先检查恢复前自动生成的安全备份文件
